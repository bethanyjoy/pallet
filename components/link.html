Index

Link

A Link takes users to another place.

Default

Use for text links inside a paragraph or for standalone text. Default links open in the same browser tab.

Underline hidden

Use when wanting to draw less attention to the link.

Plus

Use when wanting to create more visual emphasis on link.

Monochrome

Use for text links that are the same color as the surrounding text.

Size

Use for text links that are paired with size 200 paragraph text.

Props

  • url string

    The url to link to

  • text string

    The text to display inside of the link

  • external boolean

    Makes the link open in a new tab

  • monochrome boolean

    Makes the link color the same as the current text color

  • removeUnderline boolean

    Removes text decoration underline to the link

  • weight 'normal' | 'medium'

    Changes the weight of the link text

  • size '100' | '200'

    Changes the weight of the link text
